# How to open
To open the file you must have an IDE that supports C# (i recommend Visual Studio 2022).
> [!IMPORTANT]
> 1 - Download the file (or clone the repo);<br />
> 2 - Open "Quiz.sln" with the IDE of your choosing;<br />
> 3 - Hit the "Start" button.
